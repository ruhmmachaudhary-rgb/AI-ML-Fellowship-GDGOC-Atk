Movie Recommendation System (TMDB Dataset)

This repository contains a mini machine learning project focused on predicting movie popularity using the TMDB 5000 Movies Dataset.

Overview

The goal of this project is to explore features such as budget, runtime, and popularity to predict whether a movie is likely to be popular. It also includes basic exploratory data analysis (EDA), data preprocessing, and a Random Forest classifier.

Repository Structure
Movie_Recommendation.ipynb – Jupyter Notebook with full EDA, model training, and evaluation.
data/ – Folder containing the TMDB CSV datasets (tmdb_5000_movies.csv and tmdb_5000_credits.csv).
README.md – This file with project overview and instructions.
Key Features
Data cleaning and preprocessing
Feature engineering for movie popularity
Random Forest classifier to predict popular movies
EDA including genre analysis, budget vs popularity, and runtime distribution
Insights and visualizations
Dataset
TMDB 5000 Movies Dataset (CSV format)
Kaggle link
Usage

Clone the repository:

git clone <your-repo-link>

Install required packages:

pip install pandas numpy matplotlib seaborn scikit-learn
Open Movie_Recommendation.ipynb in Jupyter or Colab and run the notebook.
